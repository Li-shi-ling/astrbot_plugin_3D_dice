# Changelog

## 2.0.0

- Added a Python-first physical dice GIF generator for D4, D6, D8, D10, and D20.
- Added PyBullet headless rigid-body simulation, Pillow software rendering, and imageio GIF encoding.
- Added deterministic seeds, final orientation result detection, bounded cache cleanup, tests, and a smoke generation script.
- Removed the runtime need for Playwright, Chromium, Node.js, browser bundles, and generated frontend assets.
